# HandWritingGeometryView
HandWritingGeometryView是一款功能强大的手写控件；通过它不仅可以画出基本的线条类型，还可以画出可编辑状态的几何图形； 方便对几何图形作调整。

任何线条类型（包括几何图形）都是可以使用橡皮擦进行擦除；也可以保存在本地，方便下次展示；   

//暂时支持如下线条类型---   

RUBBER(-1),//皮擦   

CURVE(0),//曲线 

LINE(1), //直线 

DASH(2), //点曲线

 ARROW(3), //箭头 

DASHLINE(4), //点直线 

TRIANGLE(5), //三角形 

RECTANGLE(6), //矩形 

TRAPEZIUM(7), //梯形 

OVAL(8), //椭圆 

COORDINATE(9), //坐标系 

NUMBER_AXIS(10); //数轴
