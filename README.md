# HandWritingGeometryView
HandWritingGeometryView是一个可以画出几何图形的手写控件；
支持笔迹的逐条擦除和保存；支持几何图形的可编辑状态。  
   //暂时支持如下线条类型---
    RUBBER(-1),//皮擦
    CURVE(0),//曲线
    LINE(1), //直线
    DASH(2), //点曲线
    ARROW(3), //箭头
    DASHLINE(4), //点直线
    TRIANGLE(5), //三角形
    RECTANGLE(6), //矩形
    TRAPEZIUM(7), //梯形
    OVAL(8), //椭圆
    COORDINATE(9), //坐标系
    NUMBER_AXIS(10); //数轴
